An automatic optimisation framework for bioinformatics

biopsy optimises your bioinformatics tools and pipelines

Many computation biology tools have long running times and many parameters, and different parameter settings are optimal for different input datasets. This makes choosing the best parameters for a given tool very difficult, so that we typically have to settle for our best-guess settings, or just use the defaults. biopsy tackles this problem by reducing tools and pipelines to parameter spaces in an optimisation problem, and rapidly learning the optimal settings.
